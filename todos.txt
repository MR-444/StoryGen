StoryGen Roadmap:

High Priority (Immediate Next Steps)
------------------------------------
- **Trait-Based Logic**: Implement code that checks character traits during actions. 
  - Example: A character with "can_fly" can use "Sky" exits. 
  - Example: A character with "strong" can move "heavy" objects.
  - *Desirability: High. This makes the LLM's trait definitions functionally relevant.*
- **Inventory System**: Allow characters to `take` and `drop` WorldObjects.
  - *Desirability: High. Essential for puzzle-solving and story progression.*
- **LLM JSON Bridge**: Create an adapter to export the state as JSON and ingest actions from an LLM.
  - *Desirability: Critical. The primary goal of the project.*

Secondary Goals (Polish & Depth)
--------------------------------
- **Dialogue System**: Expand encounters to support multi-turn dialogue trees.
- **World Event Hooks**: Allow the world to trigger events (e.g., "OnEnterLocation" triggers a trap if a character has the "clumsy" trait).
- **Persistent Character History**: Log all meaningful events (not just locations) for the LLM to use as memory.
